# Jogo-da-Velha
Jogo da Velha desenvolvido em WinForms e .NET
Esta aplicação utiliza a abordagem de divisão e conquista para calcular as possiblidades do tabuleiro do jogo.
Tem o modo CO-OP, PC X PC e Player X PC
